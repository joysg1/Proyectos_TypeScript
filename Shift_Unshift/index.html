Métodos shift() y unshift()

Los métodos shift() y unshift() son operaciones fundamentales para manipular arreglos en JavaScript/TypeScript, específicamente para trabajar con elementos al inicio del arreglo.

unshift(elemento)

  • Agrega uno o más elementos al inicio
  • Devuelve la nueva longitud del arreglo
  • Complejidad: O(n) - todos los elementos se desplazan
  • Modifica el arreglo original

shift()

  • Elimina el primer elemento del arreglo
  • Devuelve el elemento eliminado
  • Complejidad: O(1) - solo se elimina el primero
  • Modifica el arreglo original

Casos de Uso Comunes

  • Colas (Queues): shift() para eliminar, unshift() para agregar (aunque push/pop es más eficiente)
  • Procesamiento en orden: shift() para procesar elementos en el orden en que llegaron
  • Manejo de buffers: unshift() para agregar nuevos elementos al inicio de un buffer
  • Reversión de orden: Combinar con push() para revertir elementos

Comparación de Métodos

Método Dirección Complejidad Retorna
unshift() Inicio ← O(n) Nueva longitud
shift() Inicio → O(1) Elemento eliminado
push() Final ← O(1) Nueva longitud
pop() Final → O(1) Elemento eliminado

Ejemplo Práctico

const cola = ['tarea1', 'tarea2', 'tarea3'];

// Agregar nueva tarea (alta prioridad)
cola.unshift('tareaUrgente');
// cola = ['tareaUrgente', 'tarea1', 'tarea2', 'tarea3']

// Procesar siguiente tarea
const tareaActual = cola.shift();
// tareaActual = 'tareaUrgente'
// cola = ['tarea1', 'tarea2', 'tarea3']
